Cafeteria Sustentável
> Projeto de desenvolvimento front-end focado em UI/UX, experiência do usuário e simulação de um sistema comercial real.

---


Visão geral do projeto

> Este projeto foi desenvolvido com o objetivo de consolidar e evoluir minhas habilidades em desenvolvimento web, com foco em criação de interfaces modernas, responsivas e com boa experiência de usuário.
A proposta consiste em um site completo de uma cafeteria fictícia, construído como se fosse um produto real no mercado, incluindo estrutura visual, funcionalidades e conceito de negócio.
Além do aspecto técnico, o projeto também carrega uma proposta de valor: uma cafeteria sustentável, que prioriza práticas ecológicas e o uso consciente de materiais descartáveis biodegradáveis.

---

Objetivo

Este projeto tem como foco:
* Praticar desenvolvimento front-end com um projeto realista
* Aplicar conceitos de UI/UX na criação de interfaces intuitivas
* Simular um sistema comercial completo

---

Funcionalidades do sistema

* Página inicial com apresentação da cafeteria
* Menu de produtos
* Sistema de login (área administrativa)
* CRUD completo de produtos (criar, editar, excluir e listar)
* Interface responsiva (mobile, tablet e desktop)

---

Conceito do negócio

A cafeteria foi idealizada com foco em sustentabilidade e responsabilidade ambiental.
O conceito do projeto simula um negócio real com: experiência do cliente e práticas ecológicas, como o uso de materiais biodegradáveis e redução de impactos ambientais.
